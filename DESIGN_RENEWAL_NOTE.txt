광석이네 집 v210 디자인 전면 리뉴얼 메모

반영 내용
1. PC통신 느낌이 아닌 1990년대 따뜻한 집/음악 아카이브 콘셉트로 홈 화면과 전체 UI를 재설계했습니다.
2. 메인 첫 화면은 실제 '집' 느낌이 나도록 거실 배경, 집 문패, 창문, 레코드 오브제, 방 메뉴 구조를 추가했습니다.
3. 메뉴는 '노래방 / 상영방 / 사진방 / 라디오방'처럼 광석이네 집 안의 방을 둘러보는 방식으로 바꿨습니다.
4. 카드, 버튼, 검색창, 관리자 탭, 상세 영역에 세피아/브라운/골드 계열의 입체감을 적용했습니다.
5. 기존 기능은 유지하고 CSS 오버라이드 중심으로 적용했습니다.
6. 불필요한 텔레콤 관련 파일은 ZIP에서 제거했습니다.
