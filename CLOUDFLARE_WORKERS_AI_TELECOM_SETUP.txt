광석이네 통신방 - Cloudflare Workers AI 전용 설정 안내

이번 버전은 저장된 회원 멘트, 무료 자동 반응, 로컬 템플릿 답변을 제거했습니다.
사용자가 메시지를 보낼 때마다 Cloudflare Worker가 Workers AI를 호출해 새 대사를 생성합니다.

1. GitHub에 업로드할 주요 파일
- index.html
- app.js
- style.css
- firestore.rules는 기존 규칙과 충돌 여부 확인 후 통신방 규칙만 합쳐 배포

2. Cloudflare에 다시 붙여넣을 파일
- cloudflare-worker-ai-telecom.js

Cloudflare 절차:
Workers & Pages → kks-telecom-ai → Edit code
기존 코드를 전부 지우고 cloudflare-worker-ai-telecom.js 전체를 붙여넣기
Save and deploy

3. Workers AI binding 확인
Settings → Bindings에서 Workers AI binding 이름이 반드시 AI 여야 합니다.

4. app.js의 Worker 주소
이미 아래 주소로 들어가 있습니다.
https://kks-telecom-ai.kos20050627.workers.dev

주의:
브라우저 주소창에서 Worker를 직접 열면 {"ok":false,"error":"POST only"}가 나올 수 있습니다. 정상입니다.
사이트 통신방에서 메시지를 입력해 테스트해야 합니다.
