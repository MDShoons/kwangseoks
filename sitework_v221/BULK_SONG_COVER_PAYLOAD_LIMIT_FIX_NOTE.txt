SONG 커버사진 일괄 수정 payload 초과 오류 수정

문제:
- 카테고리 안의 SONG 자료가 많을 때, 선택한 커버사진을 base64 데이터로 변환한 뒤 모든 SONG 문서에 반복 저장하면 Firestore batch 요청 크기 제한을 초과할 수 있었습니다.
- 이때 브라우저에 Request payload size exceeds the limit 오류가 표시되었습니다.

수정:
- SONG 커버사진 일괄 수정에서 파일을 선택한 경우, 이미지를 base64로 반복 저장하지 않고 GitHub/Cloudflare Worker 업로드를 먼저 수행합니다.
- 각 SONG 문서에는 업로드된 이미지의 짧은 URL만 저장합니다.
- 일괄 수정 batch 단위를 300개로 낮춰 안정성을 높였습니다.

주의:
- 파일 업로드 방식은 Cloudflare Worker 업로드 설정이 정상이어야 합니다.
- URL 입력 방식은 https로 시작하는 이미지 주소 사용을 권장합니다.
