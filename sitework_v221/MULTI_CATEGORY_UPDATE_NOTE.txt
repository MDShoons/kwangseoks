2026-06-01 multi category update

변경 내용:
1. 관리자 등록/수정 화면에서 자료 카테고리를 여러 개 선택할 수 있도록 변경했습니다.
   - 각 페이지 글/사진 등록
   - 영상 등록
   - 사진 등록
   - 음원 등록
   - 라디오 등록
   - 원음 글 등록

2. Firestore 저장 구조를 하위 호환 방식으로 확장했습니다.
   - 기존 필드: subCategory = 첫 번째 선택 카테고리
   - 신규 필드: subCategories = 선택한 카테고리 전체 배열
   기존에 subCategory 하나만 가진 자료도 그대로 표시·검색·필터됩니다.

3. 카테고리 필터는 선택한 카테고리가 subCategories 배열 안에 포함되어 있으면 자료를 보여주도록 변경했습니다.

4. 카드/목록/상세창/관리 목록에서 여러 카테고리가 모두 표시되도록 변경했습니다.

주의:
- 여러 카테고리 선택은 PC에서 Ctrl/Command 또는 Shift 키를 누른 상태로 선택하면 됩니다.
- 모바일 브라우저에서는 기기 UI에 따라 다중 선택 방식이 다르게 보일 수 있습니다.
