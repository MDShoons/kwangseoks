ATELIER REDESIGN v310

이번 버전은 이전처럼 색상만 조금 바꾸는 수준이 아니라, 사이트의 첫인상과 전체 UI 톤을 새로 잡은 디자인 리뉴얼 버전입니다.

반영 내용
1. 홈 첫 화면 재구성
- 기존 단순 소개형 hero를 프리미엄 아카이브형 hero로 교체
- 큰 타이틀, 설명, 바로가기 버튼, 레코드 오브젝트, 아카이브 티켓 장식 추가
- Videos/Songs/Radios/Photos 바로가기 도킹 카드 추가

2. 전체 디자인 톤 변경
- 싼티 나는 단색 갈색 버튼 느낌을 줄이고, 어두운 스튜디오 배경 + 따뜻한 종이 질감 + 골드 포인트로 재정의
- 1990년대 음악 아카이브 감성과 현대 음악 앱 UI를 섞은 방향

3. 상단 메뉴 리디자인
- 검은색 반투명 글래스 헤더
- 둥근 알약형 메뉴 버튼
- 스크롤 시에도 고급스럽게 남도록 sticky header 유지

4. 자료 카드 리디자인
- 영상/사진/글/최신자료 카드에 더 큰 라운드, 깊은 그림자, hover 상승 효과 적용
- 카테고리 배지는 세피아/골드 톤으로 통일

5. Songs/Radios 리디자인
- PC에서는 2열 음악 카드형으로 변경
- 커버, 제목, 설명, 플레이어가 더 음악 앱처럼 보이도록 조정
- 모바일에서는 1열로 자동 전환

6. 관리자 화면 정리
- 관리자 탭을 카드형 그리드로 정리
- 기능은 유지하고 시각적 정돈감만 개선

주의
- 기존 Firebase 데이터 구조와 app.js의 주요 기능은 건드리지 않았습니다.
- 디자인 오버라이드 CSS를 style.css 맨 아래에 추가했습니다.
- 캐시 방지를 위해 index.html의 CSS 버전을 atelier-redesign-v310으로 변경했습니다.
