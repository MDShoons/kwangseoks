김광석닷컴 참고형 게시판 아카이브 테마 v3

적용 내용
1. 상단 유틸리티 바 추가
   - 접속자/인기검색어 느낌의 얇은 상단바를 추가했습니다.

2. 메뉴 구조를 한국어 게시판형으로 정리
   - HOME, 사이트 소개, 영상, 음반/노래, 라디오, 갤러리, 그대에게, 소개, 둥근소리글

3. 메인 화면을 포털/게시판형으로 재구성
   - 음반/영상 대표 영역
   - 새소식/기록 목록
   - LOGIN 사이드 카드
   - 자료실 바로가기
   - 태그 영역

4. 전체 디자인 조정
   - 흰색 게시판 카드, 얇은 테두리, 브라운 계열 포인트, 사이드바 구성
   - 기존 광석이네 집의 방 배경 이미지는 아주 연하게 남겨서 정체성 유지

5. 기존 기능 유지
   - Firebase 자료 로딩, 로그인, 회원가입, 게시글형 공유 주소, 관리자 등록 기능은 그대로 유지했습니다.
