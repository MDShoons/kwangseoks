관리자 화면

글 작성, 사진 저장, 유튜브 URL 등록, 미디어 업로드/GitHub 저장, 자료 수정/삭제, 사이트 설정을 관리합니다.

각 페이지 글/사진 등록

이미지는 자동 압축 후 Firestore에 저장됩니다. Worker 없이 바로 작동합니다.

영상 등록

유튜브 URL은 바로 저장됩니다. mp4 링크 업로드는 Cloudflare Worker가 정상 연결되어야 GitHub videos 폴더에 저장됩니다. 95MB 이하만 권장합니다.

사진 등록

사진은 자동 압축 후 Firestore에 바로 저장됩니다.

음원 등록

mp3/wav 링크 업로드는 Cloudflare Worker가 정상 연결되어야 GitHub audios 폴더에 저장됩니다. 95MB 이하만 권장합니다. URL 직접 입력도 가능합니다.

라디오 등록

mp3/wav 링크 업로드는 Cloudflare Worker가 정상 연결되어야 GitHub radios 폴더에 저장됩니다. 95MB 이하만 권장합니다. URL 직접 입력도 가능합니다.
