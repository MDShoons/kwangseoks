GitHub 적용 후 확인:
1. https://mdshoons.github.io/kwangseoks/
2. Ctrl + F5
3. 사이트 하단 버전이 아래처럼 보여야 함:
   v82-http-link-proxy

Cloudflare Worker도 반드시 별도로 교체:
1. 이 ZIP 안의 cloudflare-worker-github-uploader.js 열기
2. 내용 전체 복사
3. Cloudflare → Workers & Pages → kwangseoks-uploader
4. Edit code
5. 기존 코드 전체 삭제 후 붙여넣기
6. Deploy

Worker 확인:
https://kwangseoks-uploader.kos20050627.workers.dev/health
