v83 GitHub Pages 미디어 폴더 방식

이 ZIP을 압축 풀면 바로 index.html, app.js, style.css, audios 폴더 등이 보여야 합니다.

추천 미디어 저장 위치:
- Songs 음원: audios/song-001.mp3
- Radios 음원: radios/radio-001.mp3
- Videos 영상: videos/video-001.mp4
- Photos 사진: images/photos/photo-001.jpg

사이트에 넣을 URL:
- https://mdshoons.github.io/kwangseoks/audios/song-001.mp3
- https://mdshoons.github.io/kwangseoks/radios/radio-001.mp3
- https://mdshoons.github.io/kwangseoks/videos/video-001.mp4
- https://mdshoons.github.io/kwangseoks/images/photos/photo-001.jpg

편의 기능:
Songs/Radios URL 칸에 song-001.mp3처럼 파일명만 넣으면 audios/song-001.mp3 주소로 자동 변환됩니다.
videos/photo는 확장자 기준으로 videos 또는 images/photos 폴더로 자동 연결됩니다.

중요:
GitHub 저장소 최상단에 파일과 폴더를 덮어써야 합니다.
폴더째 한 번 더 감싸서 올리면 안 됩니다.
